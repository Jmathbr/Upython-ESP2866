# uConnect-ESP2866
Projetos com Micropython usando o ESP2866
<br>
"basta colocar o arquivo "uconnect.py" dentro do esp"
<br>
Funçoes:
<br>
*from uconnect import uConnect*
<br>
cn = uConnect()

*cn.wlan_menu()*
- Menu de opçoes

*cn.wlan_manual()*
 - Connectar manualmente redes salvas
 
*cn.wlan_save()*
- Salvar uma rede

*cn.wlan_list()*
- listar redes salvas

*cn.wlan_remove()*
- remover listas salvas
